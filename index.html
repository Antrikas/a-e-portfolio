  • Time Box

    HTML, CSS, JavaScript, React, Typescript

    A user-friendly platform for organizing tasks and planning daily schedules. It provides tools for managing events through a drag-and-drop calendar system and supports day, week, and month views for easy planning.   • NFT Marketplace

    React, JavaScript, HTML, CSS

    In my NFT Marketplace project, I used React for an interactive and modular interface, with React Router enabling dynamic page loading based on NFT IDs in the URL. Custom animations enhance the user experience during loading states, while efficient state management ensures smooth data flow. The project incorporates multiple libraries and APIs to fetch and display NFT data in real-time, emphasizing responsive design for full accessibility across devices. This project demonstrates advanced frontend skills, API integration, and modern React practices.

  • E-Commerce Library

    HTML, CSS, JavaScript, React

    For my online Library project, I used React for a responsive, component-driven structure, integrating React Router for smooth navigation and Fetch API to handle data requests. I implemented custom CSS modules to achieve modular and maintainable styling, making the app fully responsive. State management was handled through React’s Context API, ensuring effective data flow across components. This project showcases essential frontend skills in modern library integration, responsive design, and efficient state management.
